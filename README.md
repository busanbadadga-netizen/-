# 치과쌤 | Dentist · Creator

치과쌤 개인 포트폴리오 웹사이트입니다.

## 구조

```
├── index.html      # 메인 HTML
├── style.css       # 스타일시트
├── js/
│   └── main.js     # JavaScript
├── images/
│   └── profile.jpg # 프로필 사진 (직접 추가)
├── assets/         # 기타 리소스
└── README.md
```

## 사용법

1. `images/` 폴더에 `profile.jpg` 프로필 사진을 넣어주세요.
2. GitHub에 push 후 Settings → Pages에서 배포하세요.

## GitHub Pages 배포

1. GitHub에 새 Repository 생성
2. 파일 전체 push
3. Settings → Pages → Source: `main` branch → Save
4. `https://<username>.github.io/<repo-name>/` 에서 확인
